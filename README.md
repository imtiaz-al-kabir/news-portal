## **Project Description**
**News Portal** is a fullstack application built with **Next.js 13**, **React**, **Tailwind CSS**, and **Node.js/Express**.  
It allows admins to **create, view, and delete news articles**, while users can **browse, filter, and read news** by category.  

This project demonstrates a clean, modern UI, dynamic news loading, server-side rendering for SEO, and secure admin routes with **Clerk** authentication.

---

## **ðŸ“Œ Features**
- Dynamic **news cards** with images, title, snippet, and categories  
- **Auto-sliding carousel** on the homepage  
- Admin panel to **create and delete news items**  
- **News detail page** with publication date, source, and full description  
- Responsive design for **mobile, tablet, and desktop**  
- Client-side deletion with **confirmation prompt**  
- Integration with **MongoDB** backend
